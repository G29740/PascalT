# Pascal's Triangle
Java implementation of Pascal's Triangle (https://en.wikipedia.org/wiki/Pascal%27s_triangle) as an academic project.

## How to install 
Clone this git using your favourite IDE ("Team > Git > Clone" in Netbeans, "File > New > Project from Version Control" in IntelliJ, "File > Importâ€¦ > Git > Projects from Git" in Eclipse, etc) using the URL `https://github.com/G29740/PascalT`

## How to use 
Simply run `main.java` to display the triangle. The number of rows can be changed with a value from 1 to 25.
